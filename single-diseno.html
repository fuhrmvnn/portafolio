CONTEXTO DEL
PROYECTO

El proyecto buscaba reorganizar una plataforma de capacitación digital con problemas de navegación, estructura de contenido y experiencia responsive.

El sistema fue pensado para estudiantes, administradores y empresas, priorizando claridad visual y facilidad de uso.

ANTES

Interfaz sobrecargada y navegación difícil de escanear.

DESPUÉS

Sistema modular con jerarquías visuales mucho más claras.

PROBLEMAS
DETECTADOS

La plataforma había crecido sin un sistema visual consistente, generando fricción en navegación, lectura y experiencia responsive.

  • Navegación compleja y demasiados niveles visuales.

    La estructura acumulaba demasiados niveles y accesos secundarios, dificultando encontrar contenido rápidamente.

  • Componentes visuales sin sistema ni consistencia.

    Tipografías, tamaños y componentes competían entre sí, reduciendo claridad visual dentro del sistema.

  • Experiencia móvil limitada y poco optimizada.

    No existían reglas visuales unificadas entre dashboards, cursos y módulos internos.

  • Mala legibilidad y jerarquía tipográfica inconsistente.

    La interfaz no respondía correctamente en pantallas pequeñas, afectando lectura y navegación táctil.

SISTEMA
VISUAL

El nuevo sistema fue diseñado para mantener consistencia entre dashboards, cursos, perfiles y vistas responsive.
